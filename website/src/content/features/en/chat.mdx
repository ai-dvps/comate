Run multiple sessions in parallel with persistent state that survives workspace switches and reconnections. Messages render Markdown, syntax-highlighted code blocks, collapsible tool calls, and reasoning blocks. Timestamps, in-session search, activity sorting, archive filters, and Work-in-Progress tags help manage long-running threads.
