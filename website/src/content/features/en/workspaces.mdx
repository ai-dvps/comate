Each workspace is bound to a local folder and independently maintains sessions, provider settings, skills, MCP servers, and hooks. Chrome-style tabs support scrolling and quick switching; tabs and session rows show live status indicators like streaming, needs-me, and unread. The status bar displays the current Git branch and workspace path.
