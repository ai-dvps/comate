Supports both WeCom and Feishu channels; each user automatically gets a persistent session on first message. Bot sessions appear as read-only history viewers in the GUI. Settings provides a Bot Management page with Owner/Admin/Normal roles, channel ownership, member management, and audit logs. Approvals and questions are delivered as interactive cards; users control sessions with `/status`, `/stop`, `/new`, `/clear`, and `/resume`.
