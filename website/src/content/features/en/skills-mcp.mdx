Install, update, enable, or disable Claude Code plugins across user/project/local scopes with the Plugin Manager. Configure MCP servers per workspace, and discover SKILL.md bundles from skills.sh or a URL on the Skills page. The wecom CLI integration lets sessions call document, smartsheet, and messaging operations directly.
