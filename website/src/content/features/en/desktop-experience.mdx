Native desktop packaging with macOS title-bar integration and traffic-light inlay. Closing the window hides the app to the system tray so sessions keep running. The auto-updater checks GitHub Releases and downloads updates in the background. The UI supports system-aware dark/light themes and is localized in English and Simplified Chinese. Notification sounds alert you to pending requests and completed turns.
