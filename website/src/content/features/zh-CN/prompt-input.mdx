输入框随内容自动增高，支持 `/` 触发 Skill/命令选择器和 `@` 触发文件引用；提示历史可模糊搜索并快速回填；可切换 Enter 发送或 Ctrl/Cmd+Enter 发送，并保护 IME 组合输入。
