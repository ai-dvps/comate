每个工作空间绑定一个本地文件夹，独立维护会话、Provider 设置、Skills、MCP Servers 与 Hooks。Chrome 风格的标签页支持滚动和快速切换；标签与会话行上实时显示 streaming、needs-me、unread 等状态；状态栏展示当前 Git 分支与工作区路径。
